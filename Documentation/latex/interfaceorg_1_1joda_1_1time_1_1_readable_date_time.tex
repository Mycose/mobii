\subsection{Detailed Description}
Defines an instant in time that can be queried using datetime fields. 

The implementation of this interface may be mutable or immutable. This interface only gives access to retrieve data, never to change it. 

Methods in your application should be defined using {\ttfamily \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_date_time}{Readable\-Date\-Time}} as a parameter if the method only wants to read the datetime, and not perform any advanced manipulations.

Get the weekyear field value. 

The weekyear is the year that matches with the week\-Of\-Weekyear field. In the standard I\-S\-O8601 week algorithm, the first week of the year is that in which at least 4 days are in the year. As a result of this definition, day 1 of the first week may be in the previous year. The weekyear allows you to query the effective year for that day.
