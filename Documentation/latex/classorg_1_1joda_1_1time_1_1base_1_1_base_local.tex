\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_local}{Base\-Local} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_partial}{Readable\-Partial} that use a local milliseconds internal representation. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_partial}{org.\-joda.\-time.\-Readable\-Partial} interface should be used when different kinds of partial objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_partial}{Base\-Partial} subclasses may be mutable and not thread-\/safe.
