\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_date_time}{Base\-Date\-Time} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_date_time}{Readable\-Date\-Time} that stores data in {\ttfamily long} and {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_chronology}{Chronology}} fields. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_date_time}{Readable\-Date\-Time} interface should be used when different kinds of date/time objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_date_time}{Base\-Date\-Time} subclasses may be mutable and not thread-\/safe.
