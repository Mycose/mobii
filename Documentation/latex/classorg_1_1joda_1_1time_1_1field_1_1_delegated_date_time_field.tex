\subsection{Detailed Description}
{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_delegated_date_time_field}{Delegated\-Date\-Time\-Field}} delegates each method call to the date time field it wraps. 

\hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_delegated_date_time_field}{Delegated\-Date\-Time\-Field} is thread-\/safe and immutable, and its subclasses must be as well.

Adds a value (which may be negative) to the millis value, overflowing into larger fields if necessary. 

The value will be added to this field. If the value is too large to be added solely to this field, larger fields will increase as required. Smaller fields should be unaffected, except where the result would be an invalid value for a smaller field. In this case the smaller field is adjusted to be in range. 

For example, in the I\-S\-O chronology\-:\par
 2000-\/08-\/20 add six months is 2001-\/02-\/20\par
 2000-\/08-\/20 add twenty months is 2002-\/04-\/20\par
 2000-\/08-\/20 add minus nine months is 1999-\/11-\/20\par
 2001-\/01-\/31 add one month is 2001-\/02-\/28\par
 2001-\/01-\/31 add two months is 2001-\/03-\/31\par

Adds a value (which may be negative) to the partial instant, throwing an exception if the maximum size of the instant is reached.
