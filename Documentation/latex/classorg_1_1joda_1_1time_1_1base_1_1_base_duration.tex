\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_duration}{Base\-Duration} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_duration}{Readable\-Duration} that stores data in a {\ttfamily long} duration milliseconds field. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_duration}{Readable\-Duration} interface should be used when different kinds of duration objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_duration}{Base\-Duration} subclasses may be mutable and not thread-\/safe.
