\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_single_field_period}{Base\-Single\-Field\-Period} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_period}{Readable\-Period} that manages a single duration field, such as days or minutes. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_period}{Readable\-Period} interface should be used when different kinds of period objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_single_field_period}{Base\-Single\-Field\-Period} subclasses may be mutable and not thread-\/safe.

Creates a new instance representing the number of complete standard length units in the specified period. 

This factory method converts all fields from the period to hours using standardised durations for each field. Only those fields which have a precise duration in the I\-S\-O U\-T\-C chronology can be converted. 
\begin{DoxyItemize}
\item One week consists of 7 days. 
\item One day consists of 24 hours. 
\item One hour consists of 60 minutes. 
\item One minute consists of 60 seconds. 
\item One second consists of 1000 milliseconds. 
\end{DoxyItemize}\hyperlink{classorg_1_1joda_1_1time_1_1_months}{Months} and \hyperlink{classorg_1_1joda_1_1time_1_1_years}{Years} are imprecise and periods containing these values cannot be converted.
