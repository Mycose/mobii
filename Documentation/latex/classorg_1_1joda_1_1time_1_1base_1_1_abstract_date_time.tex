\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_abstract_date_time}{Abstract\-Date\-Time} provides the common behaviour for datetime classes. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_date_time}{Readable\-Date\-Time} interface should be used when different kinds of date/time objects are to be referenced. 

Whenever you want to implement {\ttfamily \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_date_time}{Readable\-Date\-Time}} you should extend this class. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_abstract_date_time}{Abstract\-Date\-Time} subclasses may be mutable and not thread-\/safe.
