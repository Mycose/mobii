\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1chrono_1_1_base_chronology}{Base\-Chronology} provides a skeleton implementation for chronology classes. Many utility methods are defined, but all fields are unsupported. 

\hyperlink{classorg_1_1joda_1_1time_1_1chrono_1_1_base_chronology}{Base\-Chronology} is thread-\/safe and immutable, and all subclasses must be as well.
