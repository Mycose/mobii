\subsection{Detailed Description}
Compiles Olson Zone\-Info database files into binary files for each time zone in the database. \hyperlink{classorg_1_1joda_1_1time_1_1tz_1_1_date_time_zone_builder}{Date\-Time\-Zone\-Builder} is used to construct and encode compiled data files. \hyperlink{classorg_1_1joda_1_1time_1_1tz_1_1_zone_info_provider}{Zone\-Info\-Provider} loads the encoded files and converts them back into \hyperlink{classorg_1_1joda_1_1time_1_1_date_time_zone}{Date\-Time\-Zone} objects. 

Although this tool is similar to zic, the binary formats are not compatible. The latest Olson database files may be obtained \href{http://www.twinsun.com/tz/tz-link.htm}{\tt here}. 

\hyperlink{classorg_1_1joda_1_1time_1_1tz_1_1_zone_info_compiler}{Zone\-Info\-Compiler} is mutable and not thread-\/safe, although the main method may be safely invoked by multiple threads.
