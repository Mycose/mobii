\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1_date_time_constants}{Date\-Time\-Constants} is a non-\/instantiable class of constants used in the date time system. These are the I\-S\-O8601 constants, but should be used by all chronologies. 

\hyperlink{classorg_1_1joda_1_1time_1_1_date_time_constants}{Date\-Time\-Constants} is thread-\/safe and immutable.
