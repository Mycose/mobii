\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_interval}{Base\-Interval} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_interval}{Readable\-Interval} that stores data in two {\ttfamily long} millisecond fields. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_interval}{Readable\-Interval} interface should be used when different kinds of interval objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_interval}{Base\-Interval} subclasses may be mutable and not thread-\/safe.
