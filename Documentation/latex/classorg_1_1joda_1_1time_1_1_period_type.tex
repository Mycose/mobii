\subsection{Detailed Description}
Controls a period implementation by specifying which duration fields are to be used. 

The following implementations are provided\-: 
\begin{DoxyItemize}
\item Standard -\/ years, months, weeks, days, hours, minutes, seconds, millis 
\item Year\-Month\-Day\-Time -\/ years, months, days, hours, minutes, seconds, millis 
\item \hyperlink{classorg_1_1joda_1_1time_1_1_year_month_day}{Year\-Month\-Day} -\/ years, months, days 
\item Year\-Week\-Day\-Time -\/ years, weeks, days, hours, minutes, seconds, millis 
\item Year\-Week\-Day -\/ years, weeks, days 
\item Year\-Day\-Time -\/ years, days, hours, minutes, seconds, millis 
\item Year\-Day -\/ years, days, hours 
\item Day\-Time -\/ days, hours, minutes, seconds, millis 
\item Time -\/ hours, minutes, seconds, millis 
\item plus one for each single type 
\end{DoxyItemize}

\hyperlink{classorg_1_1joda_1_1time_1_1_period_type}{Period\-Type} is thread-\/safe and immutable, and all subclasses must be as well.
