\subsection{Detailed Description}
An immutable time period representing a number of seconds. 

{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds}{Seconds}} is an immutable period that can only store seconds. It does not store years, months or hours for example. As such it is a type-\/safe way of representing a number of seconds in an application. 

The number of seconds is set in the constructor, and may be queried using {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds_afdbf79be3a86ac1dc79a0ecf5fb56326}{get\-Seconds()}}. Basic mathematical operations are provided -\/ {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds_afc3766cd6194ea53d51aa7eec3b4bcde}{plus()}}, {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds_addb939c4ddbb2305b958cce073435b0e}{minus()}}, {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds_a902f92d09287bce6a9b615c0a8a2ac90}{multiplied\-By()}} and {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds_a983b104acb3ecff5354fdbb304453263}{divided\-By()}}. 

{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_seconds}{Seconds}} is thread-\/safe and immutable.
