\subsection{Detailed Description}
An immutable time period representing a number of minutes. 

{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes}{Minutes}} is an immutable period that can only store minutes. It does not store years, months or hours for example. As such it is a type-\/safe way of representing a number of minutes in an application. 

The number of minutes is set in the constructor, and may be queried using {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes_aa148401f0cf7a1ba653b04b4f3eefaa8}{get\-Minutes()}}. Basic mathematical operations are provided -\/ {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes_ac6b43175cec34a5cbbcb0357816f7df1}{plus()}}, {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes_a76b340ebf5d3798bf8976966aac560b2}{minus()}}, {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes_af5f78fa36e3956b613343c5916c05c5c}{multiplied\-By()}} and {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes_a6151e06258313cd9e7f730b3c5bb2765}{divided\-By()}}. 

{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_minutes}{Minutes}} is thread-\/safe and immutable.
