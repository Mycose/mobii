\subsection{Detailed Description}
Exception thrown when attempting to create an instant or date-\/time that cannot exist. 

Classes like
\begin{DoxyCode}
DateTime 
\end{DoxyCode}
 only store valid date-\/times. One of the cases where validity is important is handling daylight savings time (D\-S\-T). In many places D\-S\-T is used, where the local clock moves forward by an hour in spring and back by an hour in autumn/fall. This means that in spring, there is a \char`\"{}gap\char`\"{} where a local time does not exist. 

This exception refers to this gap, and it means that your application tried to create a date-\/time inside the gap -\/ a time that did not exist. Since Joda-\/\-Time objects must be valid, this is not allowed. 

Possible solutions may be as follows\-:\par
 Use {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_local_date_time}{Local\-Date\-Time}}, as all local date-\/times are valid.\par
 When converting a {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_local_date}{Local\-Date}} to a {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_date_time}{Date\-Time}}, then use {\ttfamily to\-Date\-Time\-As\-Start\-Of\-Day()} as this handles and manages any gaps.\par
 When parsing, use {\ttfamily parse\-Local\-Date\-Time()} if the string being parsed has no time-\/zone.
