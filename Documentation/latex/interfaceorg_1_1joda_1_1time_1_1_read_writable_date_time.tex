\subsection{Detailed Description}
Defines an instant in time that can be queried and modified using datetime fields. 

The implementation of this interface will be mutable. It may provide more advanced methods than those in the interface. 

Methods in your application should be defined using {\ttfamily \hyperlink{interfaceorg_1_1joda_1_1time_1_1_read_writable_date_time}{Read\-Writable\-Date\-Time}} as a parameter if the method wants to manipulate and change a date in simple ways.
