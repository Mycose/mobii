\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_period}{Base\-Period} is an abstract implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_period}{Readable\-Period} that stores data in a {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_period_type}{Period\-Type}} and an {\ttfamily int\mbox{[}\mbox{]}}. 

This class should generally not be used directly by A\-P\-I users. The \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_period}{Readable\-Period} interface should be used when different kinds of period objects are to be referenced. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_base_period}{Base\-Period} subclasses may be mutable and not thread-\/safe.

Creates a period from the given duration and end point. 

The two partials must contain the same fields, thus you can specify two {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_local_date}{Local\-Date}} objects, or two {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_local_time}{Local\-Time}} objects, but not one of each. As these are \hyperlink{classorg_1_1joda_1_1time_1_1_partial}{Partial} objects, time zones have no effect on the result. 

The two partials must also both be contiguous -\/ see \hyperlink{classorg_1_1joda_1_1time_1_1_date_time_utils_ab0b9c6c44f2dd462d5b5555177dbf481}{Date\-Time\-Utils\#is\-Contiguous(\-Readable\-Partial)} for a definition. Gets the total millisecond duration of this period relative to an end instant. 

This method subtracts the period from the specified instant in order to calculate the duration. 

An instant must be supplied as the duration of a period varies. For example, a period of 1 month could vary between the equivalent of 28 and 31 days in milliseconds due to different length months. Similarly, a day can vary at Daylight Savings cutover, typically between 23 and 25 hours.
