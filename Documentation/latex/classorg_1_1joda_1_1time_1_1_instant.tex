\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1_instant}{Instant} is the standard implementation of a fully immutable instant in time. 

{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_instant}{Instant}} is an implementation of \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_instant}{Readable\-Instant}. As with all instants, it represents an exact point on the time-\/line, but limited to the precision of milliseconds. An {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_instant}{Instant}} should be used to represent a point in time irrespective of any other factor, such as chronology or time zone. 

Internally, the class holds one piece of data, the instant as milliseconds from the Java epoch of 1970-\/01-\/01\-T00\-:00\-:00\-Z. 

For example, an \hyperlink{classorg_1_1joda_1_1time_1_1_instant}{Instant} can be used to compare two {\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1_date_time}{Date\-Time}} objects irrespective of chronology or time zone. 
\begin{DoxyPre}
boolean sameInstant = dt1.toInstant().equals(dt2.toInstant());
\end{DoxyPre}
 Note that the following code will also perform the same check\-: 
\begin{DoxyPre}
boolean sameInstant = dt1.isEqual(dt2);
\end{DoxyPre}
 

\hyperlink{classorg_1_1joda_1_1time_1_1_instant}{Instant} is thread-\/safe and immutable.
