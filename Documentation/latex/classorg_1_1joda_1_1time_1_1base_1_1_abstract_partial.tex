\subsection{Detailed Description}
\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_abstract_partial}{Abstract\-Partial} provides a standard base implementation of most methods in the \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_partial}{Readable\-Partial} interface. 

Calculations on are performed using a \hyperlink{classorg_1_1joda_1_1time_1_1_chronology}{Chronology}. This chronology is set to be in the U\-T\-C time zone for all calculations. 

The methods on this class use \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_partial_a2cbc5753aa7a7e36ebba1d8460f2809a}{Readable\-Partial\#size()}, \hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_abstract_partial_af10925b28447f9040e75ca7b9bb37c9c}{Abstract\-Partial\#get\-Field(int, Chronology)} and \hyperlink{interfaceorg_1_1joda_1_1time_1_1_readable_partial_aabea09e9a9f08ae0e594b0be9a7ee145}{Readable\-Partial\#get\-Value(int)} to calculate their results. Subclasses may have a better implementation. 

\hyperlink{classorg_1_1joda_1_1time_1_1base_1_1_abstract_partial}{Abstract\-Partial} allows subclasses may be mutable and not thread-\/safe.
