\subsection{Detailed Description}
{\ttfamily \hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_decorated_duration_field}{Decorated\-Duration\-Field}} extends \hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_base_duration_field}{Base\-Duration\-Field}, implementing only the minimum required set of methods. These implemented methods delegate to a wrapped field. 

This design allows new \hyperlink{classorg_1_1joda_1_1time_1_1_duration_field}{Duration\-Field} types to be defined that piggyback on top of another, inheriting all the safe method implementations from \hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_base_duration_field}{Base\-Duration\-Field}. Should any method require pure delegation to the wrapped field, simply override and use the provided get\-Wrapped\-Field method. 

\hyperlink{classorg_1_1joda_1_1time_1_1field_1_1_decorated_duration_field}{Decorated\-Duration\-Field} is thread-\/safe and immutable, and its subclasses must be as well.
